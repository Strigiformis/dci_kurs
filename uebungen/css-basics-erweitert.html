Zur DCI Webseite

Unsere ersten drei Pflichtaufgaben im Überblick



1 - HTML Dokument

Erstelle eine Datei namens html-dokument.html innerhalb des Ordners "aufgaben" und füge die grundlegende HTML-Struktur (head, body etc.) hinzu. Wähle ein Thema für diese Übung und ändere den Titel im head-Element entsprechend deines Themas. Füge die wichtigste Überschrift und zwei kurze Absätze ein.

Füge unterhalb der Absätze einen Link zu einer Website mit Bezug zu deinem Thema ein und oberhalb der Absätze ein Bild, das mit deinem Thema in Verbindung steht.

Zur Lösung

2 - HTML Listen

Erstelle eine Datei namens list.html im Ordner "aufgaben". Diese Datei sollte die grundlegende HTML-Struktur enthalten und füge eine ungeordnete Liste mit Artikeln hinzu, die du für dein Lieblingsrezept einkaufen müsstest. Füge eine geordnete Liste mit den Anweisungen für die Zubereitung des Rezepts hinzu.

Füge ein Bild des Lieblingsessens unter das Rezept hinzu Verlinke das Bild, um eine Website mit einem ähnlichen Rezept zu öffnen. Stelle sicher, dass das Bild angeklickt werden kann und sich die Website in einem neuen Tab zu öffnen.

Zur Lösung

3 - CSS Basics

Erstelle eine Datei mit dem Namen css-basics.html im Ordner "aufgaben". Diese Datei sollte die grundlegende HTML-Struktur enthalten. Füge 3 sections hinzu und füge in jedem dieser Abschnitte 2 Paragraphen ein.Füge ein internes Stylesheet ein.

Gestalte deine Website sinnvoll und nach Belieben. Der zweite Paragraph jeder section sollte eine andere Farbe haben als der erste Paragraph.
